# Итоговая аттестация
# Дьячков Илья Владимирович
## Описание
Приложение для генерации случайных паролей с возможностью настройки длины и выбора типов символов. Сохраняет историю сгенерированных паролей.

## Примеры использования
1. Установите необходимые библиотеки (если нужно).
2. Запустите приложение.
3. Настройте параметры и нажмите "Сгенерировать пароль".
4. Сохраните историю для дальнейшего использования.
